# vue3-template
vue3-template是vue3的项目模板，使用了vue3、vite2开发。目标是做一个开箱即用的前、中、后台前端方案。

## 特性
- 使用vue3+vite2开发
- 内置Mock数据方案(vite-plugin-mock)
- 动态路由
- 中、后台方案(master)
- 前台方案(portal)

## Build Setup
node version: >=12.0.0

vite version: >=2.0.0

* Installation dependencies
```
yarn install
```

* run
```
yarn install
```

* build
```
yarn build
```

* preview
```
yarn serve
```
